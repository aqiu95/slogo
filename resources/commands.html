SLogo Team 8 Command Reference

Basic Syntax

Token Syntax Details Examples
Constant -?[0-9]+\.?[0-9]* Real valued numbers only 36, -4.5
Variable :[a-zA-Z_]+ Any worded preceded by a colon :length, :side
Command [a-zA-Z_]+(\?)? Any word, given they are not a built-in command spin, diag
List [ ] Any succession of zero or more commands or variables, always separated by spaces [ 36, 56, 24, 109]
Comment ^#.* Lines that start with # or are empty #this is a comment

Turtle Commands

Name Details
forward pixels, fd pixels Moves the turtle in the current direction of distance pixels, returns pixels
back pixels, bk pixels Moves the turtle in the reverse of the current direction of distance pixels, returns pixels
left degrees, lt degrees Turns the turtle counter-clockwise by degrees, returns degrees
right degrees, right degrees Turns the turtle clockwise by degrees, returns degrees
setheading degrees, seth degrees Turns the turtle to degrees, returns degrees
towards x y Turns the turtle so that it faces (x, y), where (0, 0) is the center of the screen, returns degrees turned
setxy x y Sets the turtle at position (x, y), where (0, 0) is the center of the screen, returns distance moved
pendown, pd Sets the pen to start marking a trail, returns 1
penup, pu Sets the pen to not leave a trail, returns 0
showturtle, st Makes the turtle visible, returns 1
hideturtle, ht Makes the turtle invisible, returns 0
home Moves the turtle to (0, 0), returns distance moved
clearscreen, cs Removes all trails on screen and moves turtle to (0, 0), returns distance moved

Turtle Queries

Name Details
xcor Returns turtle's x-coordinate
ycor Returns turtle's y-coordinate
heading Returns turtle's heading in degrees
pendown?, pendownp Returns 1 if pen is down, 0 if it is up
showing?, showingp Returns 1 if turtle is visible, 0 if it is invisible

Math Operations

Name Details
sum x y, + x y Returns the sum of x and y
difference x y, - x y Returns the difference of x and y
product x y, * x y Returns the product of x and y
quotient x y, / x y Returns the quotient of x and y
remainder x y, % x y Returns the remainder of dividing x by y
minus x, ~ x Returns the negative of x
random max Returns a random, non-negative number that is less than max
sin degrees Returns the sin of degrees
cos degrees Returns the cos of degrees
tan degrees Returns the tan of degrees
atan degrees Returns the arctan of degrees
log x Returns the log of x
pow x y Returns x raised to the power of y
pi Returns the value of pi

Boolean Operations

Name Details
less? x y, lessp x y Returns 1 if x is less than y, otherwise 0
greater? x y, greaterp x y Returns 1 if x is greater than y, otherwise 0
equal? x y, equalp x y Returns 1 if x is equal to y, otherwise 0
notequal? x y, notequalp x y Returns 1 if x is not equal to y, otherwise 0
and x y Returns 1 if x and y are both non-zero, otherwise 0
or x y Returns 1 if x or y are non-zero, otherwise 0
not x Returns 1 if x is zero, otherwise 0

Variables, Control Structures, and User-Defined Commands

Name Details
make variable expression, set variable expression Assigns the value of expression to variable, returns expression
repeat expression [ command(s) ] Runs the command(s) expression number of times, returns value of final command executed or 0 if no command executed
dotimes [ variable limit(s) ] [ command(s) ] Runs the command(s) for each value specified in the range, returns value of final command executed or 0 if no command executed
for [ variable start end increment ] [ command(s) ] Runs the command(s) for each value specified in the range going by increment, returns value of final command executed or 0 if no command executed
if expression [ command(s) ] Runs command(s) if expression is not zero, returns value of final command executed or 0 if no command executed
ifelse expression [ true command(s) ] [ false command(s) ] Runs true command(s) if expression is not zero, else run false command(s), returns value of final command executed or 0 if no command executed
to command name [ variable(s) ] [ command(s) ] Assigns the command(s) to command name using variable(s) as parameters, returns 1 if command name is successfully define, 0 otherwise

Visual Commands

Name Details
setbackground index, setbg index 0=white, 1=red, 2=orange, 3=yellow, 4=green, 5=blue, 6=purple, 7=pink, 8=brown, 9=black, returns index
setpencolor index, setpc index 0=white, 1=red, 2=orange, 3=yellow, 4=green, 5=blue, 6=purple, 7=pink, 8=brown, 9=black, returns index
setpensize pixels Sets the current pen size to pixels, returns pixels
setshape index, setsh index Sets the current shape to the shape represented by index, returns index, 1=normal, 2=Duvall
setpalette index r g b Sets the color at index on the palette to the color represented by r g b, returns index
pencolor, pc 0=white, 1=red, 2=orange, 3=yellow, 4=green, 5=blue, 6=purple, 7=pink, 8=brown, 9=black, returns current pen color index
shape, sh Returns the index of the current shape
